¿Qué aprendí?

Durante el curso he estudiado lo siguiente

HTML

HTML, acrónimo en inglés de HyperText Markup Language («lenguaje de marcado de hipertexto»), hace referencia al lenguaje de marcado utilizado en la creación de páginas web.

CSS

CSS son las siglas en inglés de "Cascading Style Sheets" (hojas de estilo en cascada). Es un lenguaje de diseño gráfico que se utiliza para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado

JavaScript

JavaScript (JS) es un lenguaje de programación que permite a los desarrolladores crear páginas web interactivas. JavaScript se utiliza principalmente del lado del cliente, implementado como parte de un navegador web

UX/UI

UX/UI es una práctica que combina el diseño de experiencia de usuario (UX) y el diseño de interfaz de usuario (UI) para crear productos digitales. UX se refiere a la experiencia del usuario, mientras que UI se refiere a la interfaz que el usuario utiliza.
